Native decentralized exchange on the Filecoin Virtual Machine.

Jackrabbit allows users access to DeFi offerings on the FVM.

  • Token Swaps
  • Liquidity Pools
  • Decentralized Bonds

Jackrabbit utilizes the ERC-3475 tokens, bond standard for tokenized obligations with abstract on-chain metadata storage. Issuing bonds with multiple redemption data cannot be achieved with existing token standards. DeFi makes bonds more accessible to communities that historically did not have access to this asset.
